Python Interview Preparation

Key Knowledge Points

  • Core Python: Data structures (lists, dictionaries, sets, tuples), control flow (if/else, loops), functions, classes and objects (OOP basics).
  • Algorithms & Data Structures: Basic sorting (bubble, quick, merge), searching (binary, linear), common patterns like Two Pointers, Sliding Window, recursion. Understanding time and space complexity (Big O notation).
  • Pandas Library: DataFrames and Series, data loading (CSV, Excel), data cleaning (missing values, duplicates), data manipulation (groupby, merge, pivot, apply, transform), indexing and selection.
  • NumPy Library: Array operations, vectorized computations.
  • Error Handling: try-except blocks.
  • List Comprehensions and Generator Expressions: Efficient ways to create lists and iterators.

Preparation Details

  • Coding Practice: Solve problems on LeetCode (Easy to Medium), HackerRank (Python and Pandas sections).
  • Pandas Mastery: Focus heavily on Pandas for data manipulation tasks, as it's frequently tested in data science roles.
  • Explain Your Code: Be ready to explain your code, justify your choices for algorithms and data structures, and discuss time/space complexity.
  • Write Clean Code: Emphasize readability, proper variable naming, and comments where necessary.

Example 1: Find Duplicates in a List

Problem: Write a Python function to find all duplicate elements in a list.

def find_duplicates(input_list):
    seen = set()
    duplicates = set()
    for item in input_list:
        if item in seen:
            duplicates.add(item)
        else:
            seen.add(item)
    return list(duplicates)

# Test case
my_list = [1, 2, 2, 3, 4, 4, 5, 1]
print(f"Original list: {my_list}")
print(f"Duplicates: {find_duplicates(my_list)}") # Expected: [1, 2, 4] (order may vary)

Example 2: Pandas Data Aggregation

Problem: Given a Pandas DataFrame of sales data with 'Product', 'Region', and 'Sales' columns, calculate total sales per product per region.

import pandas as pd

# Sample DataFrame
data = {
    'Product': ['A', 'B', 'A', 'C', 'B', 'A', 'C'],
    'Region': ['East', 'West', 'East', 'North', 'West', 'South', 'North'],
    'Sales': [100, 150, 120, 200, 180, 90, 250]
}
df = pd.DataFrame(data)

print("Original DataFrame:")
print(df)

# Group by Product and Region, then sum Sales
total_sales_per_product_region = df.groupby(['Product', 'Region'])['Sales'].sum().reset_index()

print("\nTotal Sales per Product per Region:")
print(total_sales_per_product_region)

Example 3: Anagram Checker

Problem: Write a Python function to check if two strings are anagrams of each other.

def are_anagrams(str1, str2):
    # Normalize strings (lowercase, remove spaces)
    s1 = "".join(sorted(str1.lower().replace(" ", "")))
    s2 = "".join(sorted(str2.lower().replace(" ", "")))
    return s1 == s2

# Test cases
print(f"'listen' and 'silent': {are_anagrams('listen', 'silent')}") # True
print(f"'hello' and 'world': {are_anagrams('hello', 'world')}")   # False
print(f"'Debit Card' and 'Bad Credit': {are_anagrams('Debit Card', 'Bad Credit')}") # True
